# HR Analytics Dashboard

## Overview
This Power BI project focuses on analyzing employee attrition and workforce trends to help organizations make informed HR decisions. The dashboard transforms HR data into actionable insights by highlighting key metrics related to employee turnover, job satisfaction, and workforce demographics.

Through interactive visualizations and filters, users can explore patterns in employee attrition and better understand factors that influence workforce retention.

---

## Project Objectives
- Analyze employee attrition across the organization.
- Measure and monitor the overall attrition rate.
- Evaluate employee job satisfaction levels.
- Examine workforce distribution by age group.
- Provide HR stakeholders with data-driven insights to improve retention strategies.
